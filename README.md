# Система моніторингу кров'яного тиску

Desktop-застосунок на **Python + PyQt6** з локальним збереженням даних у **JSON**.

## Можливості
- додавання вимірювань артеріального тиску;
- облік пульсу, активності, стану користувача та приміток;
- опційне поле атмосферного тиску для співставлення показників;
- журнал вимірювань з видаленням записів;
- дашборд із короткою аналітикою;
- сторінка розширеної аналітики з графіком і висновками;
- редагування профілю користувача;
- експорт JSON-файлу.

## Запуск у PyCharm
1. Відкрити папку проєкту `bp_monitor_app`.
2. Встановити залежності:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустити файл `main.py`.

## Швидкий запуск
### Windows
Подвійний клік по `start_windows.bat`.

### Linux / macOS
```bash
./start_unix.sh
```

## Де зберігаються дані
Основна локальна база знаходиться у файлі:
`data/storage.json`

## Структура
- `main.py` — точка входу;
- `app/main_window.py` — головне вікно та сторінки;
- `app/storage.py` — робота з JSON;
- `app/analytics.py` — обчислення статистики;
- `app/widgets.py` — кастомні віджети;
- `assets/heart.svg` — іконка;
- `data/storage.json` — локальна база даних.
